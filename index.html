Rencana Proyek: Migrasi ytuploader ke Aplikasi Web

Panduan teknis untuk rekayasa ulang aplikasi desktop menjadi solusi web full-stack JavaScript.

1. Tujuan Utama

Misi: Mengganti aplikasi desktop yang ada dengan solusi web modern yang sepenuhnya ditulis dalam JavaScript, mereplikasi semua fungsionalitas inti.

Kriteria Keberhasilan:

  • Aplikasi berfungsi penuh di browser.
  • Autentikasi pengguna via Google.
  • Manajemen daftar video (tambah, edit, kelola).
  • Proses upload massal dengan progres real-time.
  • Seluruh basis kode menggunakan JavaScript/TypeScript.

2. Arsitektur & Tumpukan Teknologi

Frontend: React

  • Framework: React.js
  • Build Tool: Vite
  • Styling: Tailwind CSS
  • Komunikasi: Axios & socket.io-client

Backend: Node.js

  • Framework: Express.js
  • ORM: Prisma
  • API Google: googleapis
  • Real-time: socket.io

Database

  • Sistem: PostgreSQL
  • Model: Users, AuthTokens, Videos

3. Rencana Eksekusi Bertahap

Milestone 1: Fondasi Proyek

Inisialisasi proyek backend (Node.js) dan frontend (React), serta penyiapan environment variables.

Milestone 2: Autentikasi

Implementasi alur Google OAuth2 di backend dan pembuatan halaman login di frontend.

Milestone 3: Fungsionalitas Video

Pembuatan endpoint CRUD untuk video, UI manajemen video, dan integrasi WebSockets untuk progres upload.

Milestone 4: Pengujian & Penyelesaian

Pengujian end-to-end, penanganan error, dan penyelesaian UI/UX.
